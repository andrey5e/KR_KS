# Асинхронная платформа клиентского информирования для тюнинг-ателье

## Описание
Веб-приложение для управления заказами тюнинг-ателье с автоматическим назначением мастеров и чатом в реальном времени.

## Стек технологий
Backend: Python + FastAPI
Протоколы: HTTP/1.1 (REST API), WebSocket (чат), TCP/IP
Аутентификация: JWT-токены
БД: SQLite + SQLAlchemy
Frontend: HTML, CSS, JavaScript

## Функционал
### Администратор:
Управление мастерами и клиентами
Создание заказов с автоматическим назначением мастеров
Просмотр всех заказов

### Мастер:
Просмотр назначенных этапов
Изменение статуса выполнения
Чат с клиентом

### Клиент:
Просмотр своих заказов
Получение уведомлений в реальном времени
Чат с мастером

## Запуск проекта
Установка зависимостей:
pip install fastapi uvicorn sqlalchemy pydantic python-jose[cryptography]

## Запуск сервера:
python app.py
Сервер будет доступен по адресу: http://localhost:8000

## Сетевые протоколы
HTTP/1.1 — REST API запросы
WebSocket — Чат и уведомления в реальном времени
TCP/IP — Транспортный уровень
JWT — Сетевая аутентификация
